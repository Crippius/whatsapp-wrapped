No, unfortunately it's currently not possible to get the wanted information from your desktop, only from your phone

For every conversation uploaded onto this website, the following data will be retrieved:
  • Total number of messages
  • Average number of messages per day
  • Number of files sent
  • Most active month, weekday and time of day
  • Number of participants
  • Language used
  • And some more... see the full dataset in repo if you want to know more
    • Instead these data won't be stored:
  • The group chat's name
  • Any person's name
  • Partecipants' sensitive information (addresses, phone numbers, ecc...)
